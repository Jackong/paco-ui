## 使用说明

### 用法

- 方法一：引入CSS

```html
<link rel="stylesheet" href="//o9s0jh825.qnssl.com/paco-ui/VERSION/paco-ui.min.css" charset="utf-8">
```

> `VERSION`为版本号，具体见最新tag

- 方法二：NPM

```bash
npm i --save paco-ui
```

```js
import 'paco-ui';
//OR only button css
import 'paco-ui/css/button.css';
//OR only button postcss
import 'paco-ui/src/css/button.css';
```
