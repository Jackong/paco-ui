## 使用说明

### 用法

- 方法一：引入CSS

```html
<link rel="stylesheet" href="//o9s0jh825.qnssl.com/paco/css/paco-VERSION.min.css" charset="utf-8">
```

- 方法二：NPM

```bash
npm i --save paco-ui
```

```js
import 'paco-ui';
```
